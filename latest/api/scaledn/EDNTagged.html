EDNTagged

case class EDNTagged[A](tag: EDNSymbol, value: A) extends EDNValue with Product with Serializable

EDN tagged values look like #foo.bar/toto 123L and correspond to the extension mechanism provided by EDN. The tag implies a semantic that should be managed by a handler. By default, EDN provides 2 default handlers:

  • #inst "1985-04-12T23:20:50.52Z" for RFC-3339 instants
  • #uuid "f81d4fae-7dec-11d0-a765-00a0c91e6bf6" for UUID
